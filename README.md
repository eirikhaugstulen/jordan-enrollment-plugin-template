# Jordan Enrollment Plugin Workshop Starter

This repository is the starter template for the plugin workshop taking place in Amman from 15th to 20th November 2025.

## Getting started

```bash
git clone https://github.com/eirikhaugstulen/jordan-enrollment-plugin-template.git
cd jordan-enrollment-plugin
```

## Install dependencies

```bash
npm install
```

or

```bash
pnpm install
```

## Scripts

```bash
npm start     # start the development server
pnpm start
npm run build # produce a production bundle
pnpm run build
```
